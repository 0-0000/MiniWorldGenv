# `genv`
- 通常情况下，游戏的工作目录为程序所在目录，如 `"C:\Users\Administrator\AppData\Roaming\miniworldgameguan110"` 。  
- *数据目录*下存储着游戏数据，例如 `"C:/Users/Administrator/AppData/Roaming/miniworddata110/"` 。  
- *游戏 API id* 决定了工作目录和数据目录，例如上面的例子中，游戏 API id 为 `110` 。  
- *文档*是文件和文件夹的统称。
